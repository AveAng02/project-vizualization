
# Project Vizualization
A tool to help vizualize the representation of data structures
in computer memory.

### Graphs


```mermaid

  classDiagram
 
  
  class data_structures{
    pure virtual class
  }

    data_structures --> linear
    data_structures --> non_linear

    linear --> array
    linear --> stack
    linear --> queue
    linear --> linked_list
    linear --> heap

    non_linear --> graph
    non_linear --> tree


```
